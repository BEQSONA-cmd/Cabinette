for copy the repo please run this command:
```
git clone https://github.com/BEQSONA-cmd/Cabinette.git && cd Cabinette && ./install
```

After that you have to run:
```
cabinette
```
or
```
paulo
```
in correct directory.

for update to latest version:
```
C_update
```
